# Keyboard shortcuts

Customize under **Settings → Keyboard**. Defaults below use **Mod** = Ctrl on Windows/Linux, Cmd on macOS.

Composer **Enter** (send) and **Shift+Enter** (newline) are **not** rebindable.

## Defaults

| Command | Default | Group | When |
|---------|---------|-------|------|
| Toggle Sessions | Mod+B | Layout | Always |
| Toggle Workbench | Mod+Shift+B | Layout | Always |
| Toggle Explorer | Mod+Shift+E | Layout | Always |
| Open Settings | Mod+, | App | Not in editable field |
| Keyboard Settings | *(none)* | App | Always |
| Open Folder | Mod+O | Workspace | Not in editable field |
| Save Active File | Mod+S | Editor | Active editor tab |
| Stop Generation | Esc | Chat | While generating |
| Open Model Picker | Mod+Shift+M | Chat | Always |
| Keyboard Shortcuts help | Mod+/ | App | Always |

## Rebinding

1. **Settings → Keyboard** (or the help overlay).
2. Select a command and press the new combination.
3. **Esc** cancels; **Backspace/Delete** disables.
4. If another command already uses that chord, resolve the conflict prompt.
5. Reset individual commands or **reset all** to defaults.

## Tips

- Prefer chords that do not fight the Monaco editor or system shortcuts.
- Stop generation (**Esc**) is capture-sensitive while Settings is open — Settings takes Esc to close first.
